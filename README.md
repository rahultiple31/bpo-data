# BPO Data Entry Application

Open `index.html` in a browser to use the application.

## Features

- Select one or more source images for manual data entry.
- Type or edit all fields shown in the reference application.
- Click **Update** to save the entry in browser storage.
- Click **Show Data** to review saved rows.
- Click **Export Excel** to download `data-entry-results.xls`, which opens in Microsoft Excel.
- Click **Backup XML** to download an XML backup of all saved rows.
